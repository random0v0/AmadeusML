# KhalidTimerForML

# 메이플랜드 전용 계산기 및 타이머입니다.


F1 : 5분 타이머 시작


F2 : 1시간 타이머 시작


F3 : 초기화


F4 : 계산 (재입력 시 계산 화면 꺼짐)

Win + Shift + S로 원하는 이미지 캡쳐 후 인풋 박스에 컨트롤 + V 하면 자동 기입됩니다.
이상한 값 입력될 시 이미지 해상도 문제로, 게임 화면을 키운 후 캡처를 해주시면 됩니다.


타이머 시작 5초 전에 알림을 추가했으니, 여유롭게 대비하실 수 있습니다.
